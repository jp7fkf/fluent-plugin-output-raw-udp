## Installation

### RubyGems

```
$ gem install fluent-plugin-output-raw-udp
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-output-raw-udp"
```

And then execute:

```
$ bundle
```

